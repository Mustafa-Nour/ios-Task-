

## Features
- Login View with Phone Input
- Registration Flow
- Main View with TabBar
- Localization Support (Arabic/English)

## Project Structure
The project follows a modular structure:
- **App**: Application lifecycle and delegates.
- **Login**: Login screens and view controllers.
- **Register**: Registration screens and view controllers.
- **MainView**: Primary application interface.
- **Services**: Shared services (e.g., Authentication).
- **Utilities**: Helper classes and extensions.
